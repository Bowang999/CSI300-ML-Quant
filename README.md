# CSI300 ML Quant — Machine Learning-Based Stock Selection for the CSI 300 Index

> Final Year Project (FYP) · Bo Wang · 2026  
> Submitted to *Intelligent Systems in Accounting, Finance and Management* (ISAFM / Wiley)

---

## Overview

A complete end-to-end quantitative investment research pipeline targeting the **CSI 300 (沪深300) index**, covering raw data processing, factor engineering, multi-model training, portfolio optimisation, and backtesting.

---

## Pipeline

```
Raw Excel (CSMAR/Wind)
  └─ excel_initial_1        CSI300 constituent cleaning
  └─ excel_dailydata_2      Daily price data merging
  └─ excel_index_4          Industry code enrichment
  └─ excel_fundamental_3    Fundamental data integration
  └─ excel_dedata_5         Valuation derivative indicators
        │
        ▼
   stock_data.parquet  (not included — see Data section)
        │
  └─ Return_Calculation_MissingValue_Process
  └─ factor_construction  (+factors_engineering.py)   84 factors
  └─ factor_processing    MAD winsorise → neutralise → Z-score
  └─ monthly_return       Monthly resampling
  └─ factor_validation    IC / R² analysis
        │
        ▼
  Model Training (daily & monthly)
  └─ Ridge Regression
  └─ XGBoost
  └─ LightGBM
  └─ LSTM + Variable Selection Network (VSN)
  └─ C-ENet Ensemble
  └─ Equal-Weight Ensemble
        │
        ▼
  MVO Portfolio Optimisation → Backtesting
  └─ backtest / backtest - monthly
  └─ backtest_model / backtest_model - monthly
```

## Factor Library (84 Factors)

| Category | Examples |
|---|---|
| Momentum | 2d / 5d / 10d / 20d return, reversal, price position |
| Valuation | PE, PB, PCF, PS and their changes |
| Quality | ROE, ROA, gross margin, asset turnover, cash flow ratios |
| Risk / Volatility | Historical vol, Beta, idiosyncratic vol, illiquidity |
| Sentiment / Technical | Turnover rate, price-volume divergence, up-volume ratio |

Factor processing: **MAD winsorisation → industry + market-cap neutralisation → cross-sectional Z-score**

## Data

Raw data comes from **CSMAR** and **Wind** (commercial licences required) and is **not included** in this repository.

To reproduce the full pipeline, obtain the following datasets and place them under `RAW_DATA/`:

| Subfolder | Content |
|---|---|
| `全股票日行情 - 后复权/` | Daily stock prices (backward-adjusted), 4 xlsx files |
| `基本面数据/` | Fundamentals: 盈利能力, 经营能力, 发展能力, 现金流分析 |
| `股票衍生指标/` | Valuation multiples: PE, PB, PCF, PS (8 xlsx files) |
| `行业分类/总行业/` | Industry classification (Shenwan) |
| `沪深300指数收益率/` | CSI300 index return series |

Intermediate parquet files (generated by running the notebooks in order) are also excluded due to size (~2.8 GB total).
